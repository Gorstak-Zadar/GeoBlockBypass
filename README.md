# 🌍 GeoBlockBypass

> **Geoblocking Bypass Detection** - Detects tools and techniques used to bypass geographic restrictions.

---

## 📋 Overview

GeoBlockBypass monitors for VPN, proxy, and other tools used to bypass geographic content restrictions, useful for compliance monitoring.

---

## 🎯 What It Detects

- 🌍 **VPN Usage** - VPN connection detection
- 🌐 **Proxy Detection** - Proxy server usage
- 🔄 **Tor Detection** - Tor network usage
- 📍 **Location Spoofing** - Fake location attempts

---

## 🚀 Usage

```powershell
# Detect geo-bypass tools
.\GeoBlockBypass.ps1
```
